# BHTPA Logo

This directory should contain the official BHTPA logo file.

## Required File

- **Filename**: `logo.png`
- **Format**: PNG
- **Recommended Size**: 120x80 pixels (or higher resolution with same aspect ratio)
- **Source**: Extract from the official BHTPA documentation or website

## Instructions

1. Obtain the official BHTPA logo from:
   - Official BHTPA website (www.bhtpa.gov.bd)
   - Official documentation
   - Contact BHTPA administration

2. Save the logo as `logo.png` in this directory

3. The logo will be automatically embedded in invoice PDFs

## Note

If the logo file is not present, invoices will be generated without the logo image.
The template is designed to handle the missing logo gracefully.
